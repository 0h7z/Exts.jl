Exts.invsqrt — Method
invsqrt(x::T) -> float(T) where T <: Real

Return $\sqrt{x^{-1}}$.

See also sqrt.

Examples

julia> invsqrt(4)
0.5
source
Exts.lmap — Function
lmap(f, iterators...)

Create a lazy mapping. This is another syntax for writing (f(args...) for args in zip(iterators...)).

Examples

julia> collect(lmap(x -> x^2, 1:3))
3-element Vector{Int64}:
 1
 4
 9
